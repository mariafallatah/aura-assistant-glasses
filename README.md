# Aura – Assistant Glasses
A wearable smart glasses system designed to assist visually impaired 
students at PMU with independent campus navigation.

## Features
- YOLOv8 real-time object detection (person, chair, door)
- UWB indoor positioning with centimeter-level accuracy
- Voice interaction using Vosk + eSpeak TTS
- Tesseract OCR for reading campus signs
- Emergency panic button with live mobile app stream

## Hardware
Raspberry Pi 4B, HC-SR04 ultrasonic sensors, ESP32, UWB modules

## Technologies
Python, YOLOv8, OpenCV, Vosk, Tesseract OCR, React Native
